# pandownload-login
本地代理修复 Pandownload 登录失败

[加入 Telegram 交流群](https://t.me/pandown)
# 使用方法
-  [下载 Release](https://github.com/TkzcM/pandownload-login/releases/latest)  

    解压并运行 pdproxy.exe

- 修改 PanData/config.ini

  proxy= 后填 http://127.0.0.1:8888
- 打开 Pandownload

- 运行 proxyset.exe 修复内存泄漏
  
  Enjoy!
